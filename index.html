Wiset

A Linux Tool that helps you to change MAC address of various devices to perform network attacks. Wiset is specially designed for wireless adapters to change their MAC addresses to hide their original MAC while performing wireless attacks.

The MAC Changer

Change the MAC address of the device easily using Wiset on the go. using simple command line. This python script helps you to 

root@kali:~# python wiset.py -i <interface> -m <new_mac>

Set your wireless adapter 

Change the MAC address of the device easily using Wiset on the go. using simple command line.

root@kali:~# python wiset.py --interface wlan0 --mac 00:11:22:33:44:55

Included Modules

sys  - System specific parameters and functions
Re - Regular Expression operations
Subprocess  - Subprocess management
optparse  - parser for command line
